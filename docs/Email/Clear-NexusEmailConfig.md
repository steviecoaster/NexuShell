# Clear-NexusEmailConfig

## SYNOPSIS

Clears and disables the Smtp configuration in Nexus

## SYNTAX

```powershell
Clear-NexusEmailConfig [-Force] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION

Clears and disables the Smtp configuration in Nexus

## EXAMPLES

### EXAMPLE 1

```powershell
Clear-NexusEmailConfig
```

### EXAMPLE 2

```powershell
Clear-NexusEmailConfig -Force
```
