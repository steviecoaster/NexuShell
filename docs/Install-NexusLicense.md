# Install-NexusLicense

## SYNOPSIS
Installs a license on a nexus instance.

## SYNTAX

```
Install-NexusLicense [-NexusLicense] <String> [<CommonParameters>]
```

## DESCRIPTION
Installs a license on a nexus instance.

## EXAMPLES

### EXAMPLE 1
```
Install-NexusLicense -NexusLicense 'C:\temp\sonatype-repository-manager.lic'
```

## PARAMETERS

### -NexusLicense
